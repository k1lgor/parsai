# Keybindings and commands for PARSAI

# i3 (window manager)
• Windows key, Linux key, Alt key, ... :
Mod

• Full keybinding list :
Mod + `

• Start terminal :
Mod + Return

• Kill focused window :
Mod + Q

• Application menu :
Mod + D

• Window switcher menu :
Mod + T

• Power menu :
Mod + Shift + E

• Lock the system :
Mod + Backspace

• Change workspace:
Mod + 1/2/3/4/...
Mod + Tab

• Change window focus : 
Mod + H : Focus left
Mod + J : Focus down
Mod + K : Focus up
Mod + L : Focus right

• Alternatively, you can use the cursor keys :
Mod + Left : Focus left
Mod + Down : Focus down
Mod + Up : Focus up
Mod + Right : Focus right

• Move a focused window :
Mod + Shift + H : Move left
Mod + Shift + J : Move down
Mod + Shift + K : Move up
Mod + Shift + L : Move right

• Alternatively, you can use the cursor keys :
Mod + Shift + Left : Move left
Mod + Shift + Down : Move down
Mod + Shift + Up : Move up
Mod + Shift + Right : Move right

• Enter fullscreen mode for the focused container :
Mod + F

• Change container layout (stacked, tabbed, toggle split) : 
Mod + S : Layout stacking
Mod + W : Layout tabbed
Mod + E : Layout toggle split

• Toggle tiling / floating :
Mod + Shift + Space

• Change focus between tiling / floating windows :
Mod + Space

• Focus the parent container :
Mod + A

• Resize floating window :
Mod + Right mouse

• Move floating window :
Mod + Left mouse

• Reload the configuration file :
Mod + Shift + C

• Restart i3 : 
Mod + Shift + R

• Redirect sound to headphones :
Mod + P

• Take screenshot:
PrtScn

# Ranger (file manager)
• Copy, cut and paste :
:copy : Copy file
:paste : Paste file
:cut : Cut file

• Delete file :
:delete

• Rename file :
:rename name_of_file

• Drag and drop :
Ctrl + D

• Quit ranger :
:q

# Firefox (browser)
• New tab :
Ctrl + T

• New window :
Ctrl + N

• New private window :
Ctrl + Shift + P

• Change tab :
Ctrl + Tab

# Vim (text editor)
• Quit vim
:q if you saved file
:q! if you don't want to save file
:wq : save file and quit vim

• Save file
:w

• Enter insert mode:
I

• Back to normal mode:
Esc

• Move cursor :
H : Move left
J : Move down
K : Move up
L : Move right

• Alternatively, you can use the cursor keys :
Left : Move left
Down : Move down
Up : Move up
Right : Move right
